# CEellipse
R function to draw a confidence ellipse for cost-effective data, using either person-level or summary cost-effectiveness data

# Files
Example_Program.R: Example program demonstrating how to draw a confidence ellipse for cost-effective data, using either person-level or summary-level cost-effectiveness data. The program uses CEellipse() function (stored in CEellipse.R file)

CEellipse.R: Function to draw a 95% (or other level) confidence ellipse for cost-effective data, using either person-level or summary cost-effectiveness data

CEellipse.pdf: Help file for CEellipse() function
